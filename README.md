# Análise de Dados com Python

## 📌 Descrição

Este projeto apresenta uma análise exploratória de dados realizada em um notebook do Jupyter. O objetivo foi aplicar técnicas de manipulação e análise de dados utilizando bibliotecas da linguagem Python.

## 📊 Atividades realizadas

Durante a análise foram realizadas as seguintes etapas:

* Importação e preparação do conjunto de dados
* Exploração inicial dos dados
* Cálculo de estatísticas descritivas (média, mediana, desvio padrão, etc.)
* Identificação de valores mínimos, máximos e moda
* Análise da distribuição dos dados
* Visualização de informações por meio de gráficos

## 🛠 Bibliotecas utilizadas

As seguintes bibliotecas foram utilizadas no desenvolvimento da análise:

* **pandas** – manipulação e análise de dados
* **numpy** – operações numéricas e suporte a estruturas de dados
* **matplotlib** – criação de gráficos e visualizações
* **seaborn** – visualização estatística de dados
* **datetime** – manipulação de datas e horários

## 📂 Estrutura do repositório

* `analise_dados.ipynb` — Notebook contendo todo o processo de análise dos dados.

## ▶️ Como executar

1. Fazer o download do notebook.
2. Abrir o arquivo no Jupyter Notebook.
3. Executar as células sequencialmente para reproduzir a análise.

## 📎 Observação

Este projeto foi desenvolvido para fins de estudo e prática de análise de dados em Python.
